Welcome to Journal API

Allowed HTTP requests:
  • GET
  • POST
  • PUT
  • DELETE
Collections
Users

Routes:

  • /auth/signup POST request to register new users
  • /auth/login POST request to login existing users
Notes

Routes (cannot be accessed without user authentication):

  • /api/note GET all notes created by user
  • /api/note/:noteId/:userId GET selected note created by user
  • /api/note POST a new note created by user
  • /api/note/:noteId/:userId PUT (update) a note created by user
  • /api/note/:noteId/:userId DELETE a note created by user
